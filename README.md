## 📂 Projects & Topics Covered

### 1. Simple Linear Regression
* **Description:** Built a baseline model using Scikit-Learn's diabetes dataset to predict disease progression one year after baseline using a single feature (BMI).
* **Key Learnings & Insights:**
  * **Positive Correlation:** There is a clear upward trend showing that higher BMI is linked to increased diabetes progression.
  * **Moderate Predictive Power:** While the linear regression line safely captures the general trend, the variance in data points proves that BMI alone doesn't tell the whole story—other biological variables are necessary for accurate predictions.
  * **Data Scaling:** Learned how to interpret mean-centered and scaled data, as the dataset's features represent deviations from the average rather than raw numbers.
